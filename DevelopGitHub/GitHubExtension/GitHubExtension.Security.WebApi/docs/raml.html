    /note post

    post

    Create note about chosen collaborator

    post /note

    Create note about chosen collaborator

    • Request
    • Response

    Body

    Type: application/json

    Schema:

    {
  "type": "object",
  "required": true,
  "properties": {
    "UserId": {
        "type": "string",
        "required": false
    },
    "CollaboratorId": {
        "type": "string",
        "required": false
    },
    "Body": {
        "type": "string",
        "required": true
    }
  }
}

    Example:

    {
  "UserId": "5",
  "CollaboratorId": "100",
  "Body": "Great coder!"
}

    HTTP status code 200

    HTTP status code 400

    Body

    Type: application/json

    Example:

    {
  "$id": "1",
  "message": "The request is invalid.",
  "modelState": {
    "$id": "2",
    "model.Body": [
      "'Body' не должно быть пусто."
    ]
  }
}
